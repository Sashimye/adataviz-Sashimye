# 🚽 Adataviz — Toilettes de Paris

Projet individuel (2 semaines) — **Ada Tech School**  
Site web affichant les **toilettes publiques de Paris** via l’[API OpenData Paris](https://opendata.paris.fr/).

🔗 [Démo](https://ton-lien.netlify.app) -  📂 Repo : `adataviz-Sashimye`

***

## ⚙️ Installation

```bash
git clone https://github.com/sammykelly/adataviz-Sashimye.git
cd adataviz-sashimye
pnpm install
pnpm dev
```

***

## 🧩 Fonctionnalités

- ✅ Fetch des données depuis l’API OpenData (`sanisette`)
- ✅ Affichage dynamique de cartes responsive
- ✅ Bouton **Voir plus / Voir moins**
- ✅ Barre de recherche avec filtrage (`where`)
- ✅ Design responsive (Flexbox & Grid)

***

## 🛠️ Stack

**HTML5** -  **CSS3** -  **JavaScript (ES6+)** -  **Vite** -  **pnpm**

***

## 📡 API
